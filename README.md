### 必要な環境
```
✓ Node.js 16 以上
✓ npm または yarn
```

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/featlis/Mahjong-Score-Calculator.git
cd Mahjong-Score-Calculator

# 依存パッケージをインストール
npm install
```

### 開発・起動

```bash
# 開発モードで起動（ホットリロード対応）🎯
npm run start
```

> 💡 Viteの開発サーバー（ポート5173）とElectronアプリが自動的に起動します

## 🎮 使い方

### 1️⃣ 手牌の入力

```
┌─────────────────────────────────────────┐
│  手牌 [0 / 14]          14枚目が和了牌   │
│  ┌─────────────────────────────────────┐│
│  │ 🀇 🀈 🀉 🀊 🀋               │ ← クリック│
│  │ で手牌を追加                       │
│  └─────────────────────────────────────┘│
│                                         │
│  ドラ表示牌                             │
│  ┌─────────────────────────────────────┐│
│  │ 🀞                                 ││
│  └─────────────────────────────────────┘│
└─────────────────────────────────────────┘
```

**牌キーボード**をクリックして手牌を追加：
- **萬子（マンズ）**: 🀇 🀈 🀉 🀊 🀋 🀌 🀍 🀎 🀏
- **筒子（ピンズ）**: 🀋 🀌 🀍 🀎 🀏 🀐 🀑 🀒 🀓
- **索子（ソウズ）**: 🀏 🀐 🀑 🀒 🀓 🀔 🀕 🀖 🀗
- **字牌（ジハイ）**: 🀀 🀁 🀂 🀃 🀄 🀅 🀆

**操作ボタン**:
- **⌫ 1枚消す** - 最後に追加した牌を削除
- **✖ すべて消す** - 入力した牌をすべてリセット
- **ドラ入力モード: OFF** - ドラ表示牌の入力モードに切り替え

> 📌 手牌が14枚揃うと計算結果が表示されます

### 2️⃣ ドラ表示牌の設定

```
ドラ入力モード: OFF  ← クリック

⬇️

ドラ入力モード: ON   ← 牌キーボードでドラ牌を選択

⬇️

ドラ入力モード: OFF  ← 完了
```

> 💡 複数のドラ表示牌に対応しています

### 3️⃣ 条件の動的設定

```
┌───────────────────────────────────────────┐
│  親 / 子 (自風)                            │
│ ┌─────────────────────────────────────┐  │
│ │ [東(親)] [南(子)] [西(子)] [北(子)] │  │ ← クリック
│ └─────────────────────────────────────┘  │
│                                           │
│  ツモ / ロン                              │
│ ┌──────────────┐  ┌──────┐               │
│ │ [ツモ] [ロン]│  │場風  │ ← 動的に変更 │
│ └──────────────┘  │[東][南]              │
│                   └──────┘               │
│                                           │
│  特別役 ✓ 複数選択可                     │
│ ☑ リーチ   ☑ 一発   ☑ Wリーチ          │
│ ☑ 天和系   ☑ 海底系 ☑ 嶺上/槍槓       │
│                                           │
└───────────────────────────────────────────┘
```

**セグメンテッドコントロール**で選択：
- **親/子**: 東（親）、南・西・北（子）
- **ツモ/ロン**: あがり方を選択
- **場風**: 東、南から選択

**チェックボックス**で複数役を同時選択：
- ☑ リーチ（+1飜）
- ☑ 一発（+1飜、リーチと組み合わせ時）
- ☑ Wリーチ（ダブルリーチ、+1飜）
- ☑ 天和系（天和・地和・人和）
- ☑ 海底系（海底撈月・河底撈魚）
- ☑ 嶺上/槍槓（嶺上開花・槍槓ロン）

> ⚡ すべての条件変更が即座に計算結果に反映されます

### 4️⃣ 計算結果のリアルタイム表示

```
╔═══════════════════════════════════════════╗
║         📊 計算結果パネル                 ║
╠═══════════════════════════════════════════╣
║                                           ║
║  成立役名：3飜 30符                       ║
║                                           ║
║         ┏━━━━━━━━━━━━┓                    ║
║         ┃  5,900 点  ┃ ← 合計点数        ║
║         ┗━━━━━━━━━━━━┛                    ║
║                                           ║
║  子 2,000 / 親 3,900  ← 支払い内訳        ║
║                                           ║
╠═══════════════════════════════════════════╣
║  成立役・詳細                             ║
║  ├─ 平和（へいわ）: 1飜                 ║
║  ├─ 断幺九（たんやおちゅう）: 1飜       ║
║  └─ 三色同順（さんしょくどうじゅん）:   ║
║     2飜                                  ║
╚═══════════════════════════════════════════╝
```

**自動表示される情報**:
- ✨ **成立役名**: 満貫、跳満、倍満、3倍満、役満など
- 🎯 **合計点数**: 大きくハイライト表示
- 💰 **支払い内訳**:
  - ツモあがり: 各プレイヤーの支払い額（例：`子 1000 / 親 2000`）
  - ロンあがり: ロンあがり者の支払い額（例：`ロンあがり 12000`）
- 📋 **成立役・詳細**: すべての成立役と飜数を表示

> 🔄 手牌・ドラ・条件を変更すると自動的に更新されます

## 🔄 計算ロジック

このアプリは [riichi](https://www.npmjs.com/package/riichi) ライブラリを使用して**正確な麻雀点数計算**を実行します。

```
入力条件
  ↓
手牌 🎴 + ドラ 🏮 + 風 🧭 + 特別役 ⭐
  ↓
[riichi ライブラリ]
  ↓
✓ 成立役判定
✓ 飜数計算
✓ 符数計算
✓ 点数計算
  ↓
計算結果 📊
```

### 対応機能
- ✅ **標準麻雀ルール**に準拠
- ✅ **役満対応**: 単独役満、複合役満に対応
- ✅ **ドラ計算**: 複数ドラの加算に対応
- ✅ **切り上げ計算**: 麻雀標準の四捨五入ルールに対応
- ✅ **14飜以上**: 数え役満、役満にも対応

## 🎓 麻雀初心者向けのヒント

> 📚 **麻雀をはじめたばかりの方へ**

- **飜（ハン）**: 役の組み合わせの複雑さを表す単位。1飜から数え役満まで
- **符（フ）**: 役以外の条件（メンツ構成など）による点数加算
- **満貫**: 5飜以上で一定額（8,000点）に統一
- **役満**: 最高難度の役で、満貫の2～3倍の点数

> 💡 このアプリで計算しながら麻雀のルールを学べます！

## 🐛 トラブルシューティング

### Q: アプリが起動しない

```bash
# 1. 依存パッケージを再インストール
rm -rf node_modules package-lock.json
npm install

# 2. キャッシュをクリア
npm cache clean --force

# 3. 再度起動
npm run start
```

### Q: ビルド時にエラーが発生した

```bash
# 最新のNode.jsを使用しているか確認
node --version  # v16以上が必要

# Viteキャッシュをクリア
rm -rf dist

# ビルドを再実行
npm run dist
```

### Q: 計算結果がおかしい

- ✅ 手牌が14枚揃っているか確認
- ✅ ドラ表示牌が正しく設定されているか確認
- ✅ 選択した風が正しいか確認
