# 麻雀点数計算アプリ 使い方マニュアル

本アプリは、簡単かつ直感的な操作で麻雀の点数計算ができるWindowsデスクトップアプリケーションです。

---

## 🛠 起動方法

アプリを起動するには、以下のコマンドを実行します。

```bash
# プロジェクトのディレクトリに移動
cd c:\Users\aggsh\Documents\Mahjong-Score-Calculation

# アプリを起動する
npm run start
```
自動的にアプリケーションのウィンドウが起動します。

---

## 🎮 操作方法

画面はシンプルかつ直感的なダークモードUIで構成されています。

### 1. 条件の選択
- **親 / 子**: あがったプレイヤーが「親」か「子」かを選択します。
- **ツモ / ロン**: あがり方が「ツモあがり」か「ロンあがり」かを選択します。

### 2. 飜数・符数の選択
- **飜（ハン）**: ドロップダウンメニューから「1飜」〜「13飜以上（役満）」を選択します。
- **符（フ）**: ドロップダウンメニューから「20符」〜「110符」を選択します。
  - *※ 5飜（満貫）以上を選択した場合は、符の計算が不要になるため、自動的に符の選択欄が非表示になります。*

### 3. 計算結果の確認
- 条件を変更すると、画面下部の結果パネルに**リアルタイム**で計算結果が反映されます。
- **表示内容**:
  - あがりの名称（例: 「満貫」「2飜 30符」など）
  - 合計点数（大きく表示されます）
  - 各プレイヤーの支払い内訳（例: ロンあがりなら「ロンあがり」、ツモあがりなら「子 1000 / 親 2000」など）

---

## 📦 Windows用アプリ（.exe）のビルド方法

他のPCに配布するためのインストーラー（`.exe`）を作成したい場合は、以下のコマンドを実行してください。

```bash
npm run dist
```
ビルドが完了すると、`dist-electron/` フォルダ内にインストーラーが生成されます。
